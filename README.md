# 🫁 Tuberculosis Detection using Chest X-Ray (CXR)
### Digital Image Processing Pipeline

## 📌 Overview
This project presents a **Digital Image Processing (DIP) pipeline** for detecting **Tuberculosis (TB)** from Chest X-ray (CXR) images. The system applies image preprocessing, enhancement, feature extraction, and classification techniques to assist in early TB diagnosis.

---

## 🎯 Objectives
- Enhance chest X-ray images for better visibility
- Extract meaningful features from medical images
- Classify images as TB-positive or normal
- Build a structured pipeline for medical image analysis

---

## 🧠 Methodology

### 1. Image Preprocessing
- Grayscale conversion  
- Noise reduction (Gaussian/Median filtering)  
- Image normalization  

### 2. Image Enhancement
- Histogram Equalization  
- Contrast improvement  

### 3. Feature Extraction
- Texture features  
- Edge detection (Sobel/Canny)  
- Statistical features
